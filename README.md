# Portal automático de noticias con Python

Demo educativa que consume noticias desde Google News RSS y genera HTML automáticamente usando Python.

## Tecnologías
- Python
- Feedparser
- RSS
- HTML

## Funcionalidades
- Noticias automáticas
- Generación dinámica de HTML
- Diseño tipo portal de noticias

## Objetivo
Proyecto educativo para aprender automatización y consumo de feeds RSS.
